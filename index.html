Android Attendance App

A modern Android application for employee attendance tracking with photo capture and GPS location verification.

Features

  • Photo capture using device camera
  • GPS location tracking
  • Integration with PHP/MySQL backend
  • Modern Material Design UI
  • Proper error handling and permissions management
Download Android Studio Project

Installation Guide

  1. Extract the downloaded attendance-app.zip file
  2. Open the project in Android Studio
  3. Update the BASE_URL in Config.java with your PHP/MySQL backend URL
  4. Build and run the application
